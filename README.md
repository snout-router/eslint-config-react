## Installation

Install as a dev dependency using your preferred package manager.

## Usage

Designed for use in conjunction with [@snout-router/eslint-config].

[@snout-router/eslint-config]: https://github.com/snout-router/eslint-config

```js
// .eslintrc.cjs
module.exports = {
  extends: ["@snout/react"],
};
```
